# Target

Aplicativo mobile em Expo para acompanhamento de metas financeiras.

## Stack

- Expo
- Expo Router
- React Native
- TypeScript
- Inter Fonts

## Estrutura

- `src/app`: rotas e telas
- `src/components`: componentes reutilizaveis
- `src/theme`: cores e tipografia
- `src/utils`: utilitarios e enums

## Executar

```bash
npx expo start --clear
```

## Build local

Se houver necessidade de gerar os projetos nativos:

```bash
npx expo prebuild
npx expo run:android
npx expo run:ios
```
